# pico_half_ros
Pi:Co half v2を便利に動かすための、PC向けROSパッケージ集です。

[Pi:Co Half V2用のmicro-ROS Arduino サンプル集](https://github.com/aoki-rt/pico_half_micro_ros_arduono_examples)と組み合わせて使用します
